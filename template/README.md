## Build Setup

``` bash
# install dependencies
npm install

# 启动开发服务器：localhost:9000
npm run dev

# 打包代码
npm run build

# 发布到npm（自动打包、发布、更新版本、提交git、打tag、生成更新日志、发布）
npm run release

# 标准化提交
npm run commit

# 自动生成路由
npm run gen:entry

# 自动生成更新日志，需要 bash
npm run gen:changelog
```
